# Security Policy

## Supported Versions

| Version        | Supported |
| -------------- | --------- |
| Main           | ✅         |
| Development    | ✅         |
| Older Releases | ❌         |

## Reporting Vulnerabilities

Please do not disclose vulnerabilities publicly.

Send reports to:

[security@aurelium.io](mailto:security@aurelium.io)

Include:

* vulnerability description
* reproduction steps
* impact assessment
* proof of concept (if available)

## Response Targets

| Severity | Initial Response |
| -------- | ---------------- |
| Critical | 24 hours         |
| High     | 72 hours         |
| Medium   | 7 days           |
| Low      | 14 days          |

## Security Principles

* Zero Trust Architecture
* Least Privilege
* Secure by Default
* Continuous Verification

Thank you for helping secure AURELIUM.
